# LORD-FIGURINHAS-GEN

## Setup Instructions

To set up the LORD-FIGURINHAS-GEN project, please follow the steps below:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/lordzixz13-hash/LORD-FIGURINHAS-GEN.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd LORD-FIGURINHAS-GEN
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Set up environment variables**:
   Create a `.env` file in the root directory and add the necessary environment variables defined in `.env.example`.

5. **Run the application**:
   ```bash
   npm start
   ```

If you encounter any issues during the setup, please refer to the documentation or open an issue in the GitHub repository.
